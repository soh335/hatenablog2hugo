# hatenablog2hugo

Translating to hugo format from exported text file by Hatena Blog.

## INSTALL

```
$ go get github.com/soh335/hatenablog2hugo
```

## USAGE

```
$ hatenablog2hugo --input /path/to/file.export.txt --output /path/to/hugo/content
```
